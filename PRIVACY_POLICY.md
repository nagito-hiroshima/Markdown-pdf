# Privacy Policy

Last updated: July 1, 2026

[日本語](#日本語) / [English](#english)

## 日本語

Markdown PDF Viewer は、Markdown ファイルをブラウザ上で PDF 風に表示し、印刷または PDF 保存しやすくするための Chrome / Edge 拡張機能です。

### 収集する情報

Markdown PDF Viewer は、個人情報、閲覧履歴、アカウント情報、認証情報、位置情報、入力内容、またはユーザーを識別するための情報を収集しません。

この拡張機能は、利用状況の分析、広告、トラッキング、またはプロファイリングを行いません。

### Markdown ファイルの取り扱い

ユーザーが Markdown ファイルを開く操作をした場合、この拡張機能は指定された Markdown ファイルの内容を取得し、拡張機能内のビューアで表示します。

取得した Markdown の内容は、PDF 風の表示、印刷、またはブラウザの印刷機能による PDF 保存のためにのみ使用されます。取得した内容を開発者のサーバーや第三者のサーバーへ送信することはありません。

### ローカルファイルの取り扱い

ユーザーがローカルの Markdown ファイルを選択した場合、その内容は拡張機能内のビューアで表示するために、ブラウザ内のローカルストレージに一時的に保存されます。

ローカルファイルの内容は外部サーバーへ送信されません。

### 権限の利用目的

Markdown PDF Viewer は、以下の権限を使用します。

- `activeTab`: ユーザーが拡張機能のポップアップから現在のタブを開く操作をしたときに、現在のタブの URL を取得するために使用します。
- `storage`: ユーザーが選択したローカル Markdown ファイルの内容を、拡張機能内のビューアで表示するために一時的に保存する目的で使用します。
- ホスト権限: GitHub、GitHub Raw URL、およびユーザーが指定した Markdown URL から Markdown ファイルを取得し、ビューアで表示するために使用します。

これらの権限は、Markdown ファイルの表示、印刷、PDF 保存に必要な範囲でのみ使用されます。

### リモートコード

Markdown PDF Viewer は、外部サーバーから JavaScript などの実行可能コードを読み込んで実行しません。拡張機能で使用する JavaScript、HTML、CSS はすべて拡張機能パッケージ内に含まれています。

ユーザーが指定した URL から取得する Markdown は文書データとして扱われ、コードとして実行されません。

### 第三者への共有

Markdown PDF Viewer は、収集した情報を第三者へ販売、共有、または提供しません。そもそも、この拡張機能はユーザーを識別する情報を収集しません。

### 外部サービス

ユーザーが GitHub またはその他の Web サイト上の Markdown ファイルを開く場合、その Web サイトへのアクセスには各 Web サイトのプライバシーポリシーが適用される場合があります。

Markdown PDF Viewer は、独自の分析サービス、広告サービス、またはトラッキングサービスを使用しません。

### 変更

このプライバシーポリシーは、必要に応じて更新される場合があります。重要な変更がある場合は、このファイルの内容を更新します。

### お問い合わせ

質問や連絡がある場合は、このリポジトリの Issue からお問い合わせください。

## English

Markdown PDF Viewer is a Chrome / Edge extension that displays Markdown files in a PDF-style browser viewer and helps users print or save them as PDF files.

### Information We Collect

Markdown PDF Viewer does not collect personal information, browsing history, account information, credentials, location information, form input, or information that identifies users.

This extension does not perform analytics, advertising, tracking, or profiling.

### Handling of Markdown Files

When a user chooses to open a Markdown file, the extension fetches the specified Markdown content and displays it in the extension viewer.

The fetched Markdown content is used only to render the PDF-style preview, print the document, or save it as a PDF using the browser print feature. The content is not sent to the developer's server or to any third-party server.

### Handling of Local Files

When a user selects a local Markdown file, the file content is temporarily stored in the browser's local extension storage so that it can be displayed in the extension viewer.

Local file content is not sent to any external server.

### Permissions

Markdown PDF Viewer uses the following permissions:

- `activeTab`: Used to read the current tab URL when the user opens the current tab from the extension popup.
- `storage`: Used to temporarily store the content of a local Markdown file selected by the user so it can be displayed in the extension viewer.
- Host permissions: Used to fetch Markdown files from GitHub, GitHub Raw URLs, and Markdown URLs specified by the user, so they can be displayed in the viewer.

These permissions are used only as needed to display, print, or save Markdown files as PDF documents.

### Remote Code

Markdown PDF Viewer does not load or execute remote executable code such as JavaScript from external servers. All JavaScript, HTML, and CSS used by the extension are included in the extension package.

Markdown fetched from URLs specified by the user is treated as document data and is not executed as code.

### Sharing With Third Parties

Markdown PDF Viewer does not sell, share, or provide collected information to third parties. The extension does not collect information that identifies users.

### External Services

When a user opens a Markdown file hosted on GitHub or another website, access to that website may be subject to that website's own privacy policy.

Markdown PDF Viewer does not use its own analytics, advertising, or tracking services.

### Changes

This Privacy Policy may be updated from time to time. If important changes are made, this file will be updated.

### Contact

If you have any questions, please contact us through the Issues page of this repository.
