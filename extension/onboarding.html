Markdown PDF Viewer

Turn Markdown into a printable PDF-style page.

Open GitHub Markdown, URL Markdown, or a local file and save it as PDF from the print screen.

How to use

  1. Open a .md file on GitHub.
  2. Press the PDF View button shown on the page.
  3. You can also open URLs and local files from the extension icon.
  4. Use the print screen to save as PDF.
